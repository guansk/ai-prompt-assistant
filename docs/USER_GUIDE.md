## 快速开始

### 第一步：加载插件

1. 打开 Chrome 浏览器
2. 访问 `chrome://extensions/`
3. 开启"开发者模式"
4. 点击"加载已解压的扩展程序"
5. 选择项目根目录

### 第二步：配置飞书

1. 访问 [飞书开放平台](https://open.feishu.cn/app)
2. 创建企业自建应用
3. 记录 App ID 和 App Secret
4. 配置多维表权限
5. 创建多维表并获取 App Token 和 Table ID

### 第三步：配置插件

1. 点击插件图标
2. 点击设置按钮（⚙️）
3. 填写飞书配置信息
4. 测试连接
5. 保存配置

### 第四步：开始使用

1. 打开 ChatGPT/Claude/Gemini
2. 点击插件图标
3. 搜索或浏览 Prompt
4. 点击使用

## 安装配置

### 飞书应用配置

#### 1. 创建自建应用

访问 https://open.feishu.cn/app，创建企业自建应用：
- 应用名称：Team Prompt Manager
- 应用描述：团队 Prompt 管理工具

创建后记录：
- **App ID**（cli_ 开头）
- **App Secret**（点击"查看"获取）

#### 2. 配置权限

在"权限管理"中搜索"多维表"，开通权限：
- 推荐：仅"查看多维表"（最小权限）
- 或：完整的编辑权限

#### 3. 创建多维表

在飞书中创建多维表，添加以下字段：

| 字段名 | 字段类型 | 必填 | 说明 |
|--------|----------|------|------|
| Title | 单行文本 | 是 | Prompt 标题 |
| Content | 多行文本 | 是 | Prompt 完整内容 |
| Category | 单选 | 否 | 分类标签 |
| Status | 单选 | 否 | 状态（启用/停用） |

#### 4. 获取多维表标识

从多维表 URL 中提取：
```
https://xxx.feishu.cn/base/[App Token]?table=[Table ID]&view=xxx
```

- **App Token**：`bascn` 开头
- **Table ID**：`tbl` 开头

### 插件配置

1. 点击插件图标打开弹窗
2. 点击右上角设置按钮
3. 填写配置信息：
   - App ID
   - App Secret
   - App Token
   - Table ID
4. 点击"测试连接"验证
5. 点击"保存配置"

## 基础使用

### 打开插件

在 ChatGPT、Claude 或 Gemini 页面，点击浏览器工具栏的插件图标。

点击任意 Prompt 卡片，内容会自动填入 AI 对话框。

直接在飞书多维表中：

- 修改现有内容
- 设置分类和状态
- 停用不需要的 Prompt

## 参数化 Prompt

### 什么是参数化 Prompt？

参数化 Prompt 允许在模板中定义变量，使用时填写变量值，实现一个模板适配多种场景。

### 变量语法

| 语法 | 类型 | 说明 | 示例 |
|------|------|------|------|
| `[变量名]` | 必填 | 用户必须填写 | `[编程语言]` |
| `[?变量名]` | 选填 | 用户可以留空 | `[?代码功能]` |

### 使用步骤

1. **识别参数化 Prompt**：带有蓝色参数徽章
2. **点击 Prompt**：弹出参数填写表单
3. **填写变量**：
   - 必填项：带红色星号 ⭐
   - 选填项：灰色"(选填)"提示
4. **提交表单**：点击"生成并填入"
5. **自动填入**：完整 Prompt 填入 AI 对话框

### 示例：翻译助手

**配置：**
```
请将以下内容从 [源语言] 翻译成 [目标语言]：

专业领域：[?专业领域]

[待翻译内容]
```

**使用：**
1. 点击"翻译助手"
2. 填写：
   - 源语言：`英文`
   - 目标语言：`中文`
   - 专业领域：`医学`（选填）
   - 待翻译内容：粘贴文本
3. 点击确认

**结果：**
```
请将以下内容从 英文 翻译成 中文：

专业领域：医学

[你的文本]
```

## 常见问题

### 配置相关

**Q: 测试连接失败？**

A: 检查：
- App ID 和 App Secret 是否正确
- 应用是否已开通多维表权限
- 网络连接是否正常

**Q: Prompt 列表为空？**

A: 检查：
- 多维表中是否有数据
- Status 字段是否设置为"启用"
- App Token 和 Table ID 是否正确

### 使用相关

**Q: 点击 Prompt 没反应？**

A: 确认：
- 当前页面是支持的 AI 平台
- 页面已完全加载
- 输入框可见且可编辑

**Q: 如何更新 Prompt？**

A: 直接在飞书多维表中修改，插件会自动同步（最多 5 分钟延迟），或点击刷新按钮立即更新。

### 参数化相关

**Q: 如何识别参数化 Prompt？**

A: 参数化 Prompt 会显示蓝色徽章，如"3 个参数"。

**Q: 必填项没填会怎样？**

A: 系统会阻止提交，并显示红色边框提示。

**Q: 选填项留空会怎样？**

A: 选填项留空时，会自动替换为"无"。

## 故障排查

### 无法连接飞书 API

1. 检查 App ID 和 App Secret
2. 确认应用已开通权限
3. 检查网络连接

### 找不到输入框

1. 确认当前页面是支持的 AI 平台
2. 刷新页面后重试
3. 如页面改版，可能需要更新插件

### 数据不同步

1. 点击刷新按钮手动更新
2. 检查飞书多维表配置
3. 验证 App Token 和 Table ID
