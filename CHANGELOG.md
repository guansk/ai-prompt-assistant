## [1.2.0] - 2026-03-10

### 新增
- ✨ **支持简化的多维表链接格式**
  - 支持不带 `?table=xxx` 参数的链接（如 `https://xxx.feishu.cn/base/xxxxx`）
  - 自动获取多维表的第一个表格
  - 向后兼容完整链接格式
- ✨ **配置导出导入功能**
  - 导出配置为 JSON 文件
  - 一键导入配置
  - 方便团队成员快速配置和分享

### 改进
- 🔄 优化链接解析逻辑，支持更多链接格式
- 🎨 配置页面新增"配置管理"区域
- 📝 更新用户指南，添加新功能说明

### 文档
- 📝 更新 `USER_GUIDE.md` - 添加配置导入导出说明
- 📝 更新 `CHANGELOG.md` - 记录新版本变更

## [1.1.0] - 2024-03-05

### 新增
- ✨ **参数化 Prompt 功能**：支持在 Prompt 中定义变量占位符
  - 必填变量语法：`[变量名]`
  - 选填变量语法：`[?变量名]`
  - 动态表单生成，自动识别变量并生成输入框
  - 必填项验证，未填写时阻止提交
  - 选填项留空时自动替换为"无"
- ✨ 变量填充表单界面
  - 清晰的必填/选填标识
  - 友好的错误提示
  - 返回列表功能
- 📝 参数化 Prompt 使用指南
- 📝 测试用例和示例文档

### 改进
- 🔄 向后兼容：无变量的 Prompt 保持原有行为
- 🎨 优化表单 UI 设计
- 🎨 增强用户体验

### 文档
- 📝 `参数化Prompt使用指南.md` - 详细的中文使用指南
- 📝 `PARAMETERIZED-PROMPTS.md` - 英文快速参考
- 📝 `test-parameterized-prompt.html` - 测试用例和示例
