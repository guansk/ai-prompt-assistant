## [1.0.0] - 2024-03-05

### 新增
- ✨ 飞书 API 配置管理功能
- ✨ 多维表数据读取和缓存
- ✨ Popup 弹窗界面，展示 Prompt 列表
- ✨ 快速搜索功能，支持标题和内容模糊匹配
- ✨ 一键填充功能，支持 ChatGPT 和 Claude
- ✨ 分类标签展示
- ✨ 状态筛选（启用/停用）
- ✨ 配置页面，支持连接测试
- ✨ 智能缓存机制（5分钟过期）
- ✨ 完整的错误处理和状态提示
