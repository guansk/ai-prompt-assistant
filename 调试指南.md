# 调试指南

## 常见问题排查

### 1. 连接测试失败

#### 问题：JSON 解析错误

**错误信息示例：**
```
连接测试失败: Unexpected non-whitespace character after JSON at position 4
```

**可能原因：**
1. App ID 或 App Secret 错误
2. 飞书 API 返回了 HTML 错误页面而不是 JSON
3. 网络问题或 CORS 限制

**排查步骤：**

1. **检查配置信息**
   - App ID 格式：`cli_` 开头
   - App Secret：长字符串，无空格
   - App Token 格式：`bascn` 开头
   - Table ID 格式：`tbl` 开头

2. **查看控制台日志**
   - 右键插件设置页面 -> 检查
   - 查看 Console 标签
   - 找到 "Token response:" 或 "Table response:" 的输出
   - 查看实际返回的内容

3. **验证飞书应用配置**
   - 登录飞书开放平台
   - 确认应用状态为"已启用"
   - 确认已开通"多维表"权限
   - 重新获取 App Secret（可能过期）

4. **测试 API 可访问性**
   - 在浏览器中直接访问：`https://open.feishu.cn/open-api/auth/v3/tenant_access_token/internal`
   - 应该返回 405 Method Not Allowed（正常，因为需要 POST）
   - 如果返回其他错误，可能是网络问题

#### 问题：权限不足

**错误信息示例：**
```
连接测试失败: 权限不足
```

**解决方案：**
1. 进入飞书开放平台
2. 选择你的应用
3. 点击"权限管理"
4. 搜索"多维表"
5. 开通以下权限之一：
   - 查看多维表（推荐）
   - 查看、评论、编辑和管理多维表

#### 问题：找不到多维表

**错误信息示例：**
```
连接测试失败: 读取多维表失败
```

**解决方案：**
1. 确认 App Token 和 Table ID 正确
2. 确认多维表已分享给应用
3. 在多维表右上角点击"..."-> "高级设置" -> "应用权限"
4. 添加你的应用

### 2. Popup 弹窗问题

#### 问题：弹窗显示"未完成配置"

**解决方案：**
1. 点击"前往配置"按钮
2. 填写所有必填项
3. 点击"测试连接"验证
4. 点击"保存配置"

#### 问题：弹窗显示"加载失败"

**排查步骤：**
1. 右键插件图标 -> 检查弹出内容
2. 查看 Console 错误信息
3. 检查网络请求（Network 标签）
4. 确认配置信息正确

#### 问题：Prompt 列表为空

**可能原因：**
1. 多维表中没有数据
2. 所有 Prompt 的 Status 都是"停用"
3. Content 字段为空

**解决方案：**
1. 在飞书多维表中添加测试数据
2. 确保 Status 字段为"启用"
3. 确保 Content 字段有内容
4. 点击刷新按钮

### 3. 内容注入问题

#### 问题：点击 Prompt 没有反应

**排查步骤：**
1. 确认当前页面是 ChatGPT 或 Claude
2. 确认页面已完全加载
3. 打开页面的开发者工具（F12）
4. 查看 Console 是否有错误

**可能原因：**
1. 网站页面结构改版
2. Content Script 未正确注入
3. 输入框选择器失效

**临时解决方案：**
- 刷新页面后重试
- 手动复制 Prompt 内容

#### 问题：填充后无法发送

**可能原因：**
- 网站的 React/Vue 框架未检测到输入事件

**解决方案：**
- 填充后手动点击一下输入框
- 或在内容末尾添加一个空格

### 4. 性能问题

#### 问题：加载很慢

**优化建议：**
1. 减少多维表中的数据量（< 500 条）
2. 使用分类管理，停用不常用的 Prompt
3. 清除浏览器缓存后重试

#### 问题：搜索卡顿

**解决方案：**
- 减少 Prompt 数量
- 简化 Prompt 内容

## 调试技巧

### 1. 查看插件日志

**Popup 页面：**
```
1. 点击插件图标打开弹窗
2. 右键弹窗空白处 -> 检查
3. 查看 Console 标签
```

**Options 页面：**
```
1. 右键插件图标 -> 选项
2. 右键页面空白处 -> 检查
3. 查看 Console 标签
```

**Content Script：**
```
1. 在 ChatGPT/Claude 页面按 F12
2. 查看 Console 标签
3. 查找插件相关的日志
```

**Background（如果有）：**
```
1. 访问 chrome://extensions/
2. 找到插件
3. 点击"检查视图"
```

### 2. 查看网络请求

1. 打开开发者工具
2. 切换到 Network 标签
3. 执行操作（如测试连接）
4. 查看请求和响应

**关键请求：**
- `tenant_access_token/internal` - 获取 Token
- `bitable/v1/apps/.../records` - 读取多维表

### 3. 查看存储数据

```javascript
// 在 Console 中执行
chrome.storage.local.get(null, (data) => {
  console.log('存储的数据:', data);
});
```

### 4. 清除缓存

```javascript
// 在 Console 中执行
chrome.storage.local.remove(['cachedPrompts', 'cacheTime'], () => {
  console.log('缓存已清除');
});
```

## 获取帮助

如果以上方法都无法解决问题：

1. **收集信息**
   - 错误信息截图
   - Console 日志
   - 配置信息（隐藏敏感部分）
   - Chrome 版本
   - 操作系统

2. **创建 Issue**
   - 访问项目 GitHub 页面

   - 提供详细信息

3. **临时方案**
   - 手动复制粘贴 Prompt
   - 使用飞书多维表的分享功能

## 常用命令

### 重新加载插件
```
1. 访问 chrome://extensions/
2. 找到插件
3. 点击刷新图标
```

### 查看插件 ID
```
1. 访问 chrome://extensions/
2. 开启"开发者模式"
3. 查看插件卡片上的 ID
```

### 导出配置
```javascript
chrome.storage.local.get(['appId', 'appToken', 'tableId'], (data) => {
  console.log(JSON.stringify(data, null, 2));
});
```

### 导入配置
```javascript
chrome.storage.local.set({
  appId: 'your_app_id',
  appSecret: 'your_app_secret',
  appToken: 'your_app_token',
  tableId: 'your_table_id'
}, () => {
  console.log('配置已导入');
});
```
