# 重新加载扩展指南

## 问题已修复 ✅

CSP 问题已经解决，现在可以正常加载扩展了！

## 快速步骤

### 1. 打开扩展管理页面

在 Chrome 地址栏输入：
```
chrome://extensions/
```

或者：
- 点击右上角的扩展图标（拼图）
- 点击"管理扩展程序"

### 2. 开启开发者模式

在页面右上角，打开"开发者模式"开关。

### 3. 重新加载扩展

找到 "AI Prompt 助手" 扩展，点击"重新加载"按钮（🔄）。

### 4. 验证加载成功

✅ 应该没有红色错误提示  
✅ 扩展图标正常显示  
✅ 可以点击图标打开 popup  

### 5. 测试图标显示

1. 点击扩展图标
2. 检查是否看到：
   - 🔄 刷新按钮（右上角）
   - ⚙️ 设置按钮（右上角）
   - 🔍 搜索框图标
3. 如果看到图标，说明一切正常！

## 如果还有问题

### 检查文件结构

确认 `libs/lucide.min.js` 文件存在：

```bash
# 在项目目录运行
dir libs
```

应该看到：
```
lucide.min.js (约 398KB)
```

### 检查控制台

1. 在扩展管理页面，点击"检查视图"下的链接
2. 查看控制台是否有错误
3. 应该看到：`✅ Lucide Icons 加载成功！`

### 完全重新加载

如果还有问题，尝试：

1. 在扩展管理页面点击"移除"
2. 关闭 Chrome
3. 重新打开 Chrome
4. 重新加载扩展
