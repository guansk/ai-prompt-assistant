## 常见问题

### Q: 为什么选择 Lucide Icons？

A: 
- 轻量（60KB vs Font Awesome 300KB）
- 现代设计风格
- 完全免费商用
- 社区活跃，持续更新
- Chrome 插件友好

### Q: 可以离线使用吗？

A: 是的！本项目已经将 Lucide Icons 下载到 `libs/` 目录，完全离线可用。

### Q: 如何添加新图标？

A: 
1. 访问 https://lucide.dev/icons
2. 找到需要的图标
3. 复制图标名称
4. 在 HTML 中使用 `<i data-lucide="icon-name"></i>`
5. 调用 `lucide.createIcons()`

### Q: 图标不显示怎么办？

A: 检查：
1. 文件是否存在：`libs/lucide.min.js`
2. 是否调用了 `lucide.createIcons()`
3. 图标名称是否正确（查看官网）
4. 打开控制台查看是否有错误

### Q: 如何自定义图标样式？

A: 使用 CSS 控制：
```css
.my-icon svg {
  width: 24px;
  height: 24px;
  color: #3b82f6;
  stroke-width: 2;
}
```
