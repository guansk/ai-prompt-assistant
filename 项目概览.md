# Team Prompt Manager - 项目概览

## 📦 项目信息

- **项目名称**：Team Prompt Manager
- **版本**：1.0.0
- **类型**：Chrome 浏览器插件
- **技术栈**：Manifest V3, 原生 JavaScript, 飞书开放平台 API
- **许可证**：MIT

## 🎯 项目目标

创建一款轻量级的浏览器插件，基于飞书多维表实现团队 Prompt 的统一管理和一键填充，提升团队使用 AI 工具的效率。

## ✨ 核心功能

### 已实现功能 ✅

1. **配置管理**
   - 飞书 API 凭证配置
   - 多维表数据源配置
   - 连接测试功能
   - 本地安全存储

2. **数据管理**
   - 从飞书多维表读取数据
   - 智能缓存机制（5分钟）
   - 手动刷新功能
   - 状态筛选（启用/停用）

3. **用户界面**
   - Popup 弹窗界面
   - 快速搜索功能
   - 分类标签展示
   - 响应式设计

4. **内容注入**
   - ChatGPT 页面支持
   - Claude 页面支持
   - 一键填充功能
   - 智能 DOM 选择器

5. **状态管理**
   - 加载状态
   - 错误状态
   - 空状态
   - 未配置状态

- [ ] 使用统计

## 🎨 设计特点

### 视觉设计

- **配色方案**：蓝色主题（#3b82f6），专业简洁
- **字体**：系统默认字体栈，跨平台一致性
- **圆角**：8px 统一圆角，现代化风格
- **阴影**：轻微阴影，增加层次感

### 交互设计

- **即时反馈**：Hover、Active 状态
- **流畅动画**：0.2s 过渡动画
- **加载状态**：骨架屏、加载动画
- **错误处理**：友好的错误提示

### 响应式设计

- **固定宽度**：380px 弹窗宽度
- **自适应高度**：根据内容调整
- **滚动优化**：自定义滚动条样式

## 🔧 技术实现

### 前端技术

- **Manifest V3**：最新的 Chrome 扩展标准
- **原生 JavaScript**：无框架依赖，轻量高效
- **CSS3**：现代化样式，动画效果
- **Chrome APIs**：storage, tabs, scripting

### 后端集成

- **飞书开放平台 API**：
  - 认证：tenant_access_token
  - 数据读取：Bitable API
  - 权限：最小权限原则

### 数据流程

```
用户操作
  ↓
Popup 界面
  ↓
读取本地缓存 → 缓存有效？
  ↓ 否          ↓ 是
请求飞书 API   直接展示
  ↓
解析数据
  ↓
更新缓存
  ↓
展示列表
  ↓
用户点击
  ↓
发送消息到 Content Script
  ↓
注入目标网页
  ↓
填充输入框
```

## 🔒 安全考虑

### 数据安全

- **本地存储**：敏感信息仅存储在本地
- **HTTPS**：所有 API 请求使用 HTTPS
- **最小权限**：仅申请必要的权限

### 隐私保护

- **无追踪**：不收集用户数据
- **无上传**：不向第三方服务器发送数据
- **透明**：开源代码，可审计
