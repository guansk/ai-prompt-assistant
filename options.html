📖 配置指南

  1. 访问 飞书开放平台 创建自建应用
  2. 在应用详情页获取 App ID 和 App Secret
  3. 为应用添加"多维表"权限（仅需读取权限）
  4. 创建多维表，包含字段：Title（单行文本）、Content（多行文本）、Category（单选）、Status（单选）
  5. 从多维表 URL 中提取 App Token 和 Table ID

✨ 参数化 Prompt 使用说明 v1.1

在 Prompt 中使用变量，让一个模板适配多种场景！

变量语法：

[变量名] 必填 用户必须填写此字段
[?变量名] 选填 用户可以留空，留空时替换为"无"

示例：

在飞书多维表的 Content 字段中配置：
请将以下内容从 [源语言] 翻译成 [目标语言]：

专业领域：[?专业领域]
语气风格：[?语气风格]

[待翻译内容]
使用时：点击后会弹出表单，填写变量后自动生成完整 Prompt

💡 使用技巧：

  • 变量名要清晰明了，让用户一看就知道该填什么
  • 核心信息用必填 [变量名]，补充信息用选填 [?变量名]
  • 建议每个 Prompt 不超过 5-6 个变量
  • 无变量的 Prompt 保持原有的一键填充行为
