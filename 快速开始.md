# 快速开始指南

## 第一步：生成图标

1. 在浏览器中打开 `generate-icons.html` 文件
2. 点击每个图标下方的"下载"按钮
3. 将下载的文件重命名并保存到 `icons/` 目录：
   - icon16.png
   - icon48.png
   - icon128.png

## 第二步：加载插件

1. 打开 Chrome 浏览器
2. 在地址栏输入：`chrome://extensions/`
3. 打开右上角的"开发者模式"开关
4. 点击"加载已解压的扩展程序"
5. 选择本项目的根目录
6. 插件加载成功后，会在浏览器工具栏显示图标

## 第三步：配置飞书应用

### 3.1 创建飞书自建应用

1. 访问：https://open.feishu.cn/app
2. 登录飞书账号
3. 点击"创建企业自建应用"
4. 填写应用信息：
   - 应用名称：Team Prompt Manager
   - 应用描述：团队 Prompt 管理工具
5. 创建完成后，记录以下信息：
   - **App ID**（cli_ 开头）
   - **App Secret**（点击"查看"获取）

### 3.2 配置应用权限

1. 在应用管理页面，点击"权限管理"
2. 搜索"多维表"
3. 开通权限：
   - 推荐：仅开通"查看多维表"（最小权限）
   - 或：开通"查看、评论、编辑和管理多维表"（如需在插件中编辑）
4. 点击"保存"

### 3.3 创建多维表

1. 在飞书中创建一个新的多维表
2. 添加以下字段：

| 字段名 | 字段类型 | 必填 | 说明 |
|--------|----------|------|------|
| Title | 单行文本 | 是 | Prompt 标题 |
| Content | 多行文本 | 是 | Prompt 完整内容 |
| Category | 单选 | 否 | 分类（研发/产品/设计/运营/客服/其他） |
| Status | 单选 | 否 | 状态（启用/停用），默认：启用 |

3. 添加测试数据：

**示例数据 1：**
- Title: `代码审查`
- Content: `请审查以下代码，关注：1. 潜在的 bug 2. 安全漏洞 3. 性能问题 4. 代码规范 5. 最佳实践。请给出具体的改进建议。`
- Category: `研发`
- Status: `启用`

**示例数据 2：**
- Title: `需求分析`
- Content: `请分析以下产品需求：1. 用户痛点是什么？2. 核心价值是什么？3. 技术可行性如何？4. 潜在风险有哪些？5. 优先级建议`
- Category: `产品`
- Status: `启用`

### 3.4 获取多维表标识

1. 打开刚创建的多维表
2. 查看浏览器地址栏的 URL，格式如下：
   ```
   https://xxx.feishu.cn/base/[App Token]?table=[Table ID]&view=xxx
   ```
3. 提取并记录：
   - **App Token**：`bascn` 开头的字符串
   - **Table ID**：`tbl` 开头的字符串

## 第四步：配置插件

1. 点击浏览器工具栏的插件图标
2. 点击右上角的"设置"按钮（齿轮图标）
3. 填写配置信息：
   - App ID：从第三步获取
   - App Secret：从第三步获取
   - App Token：从第三步获取
   - Table ID：从第三步获取
4. 点击"测试连接"按钮，验证配置是否正确
5. 如果测试成功，点击"保存配置"

## 第五步：开始使用

1. 打开 ChatGPT (https://chatgpt.com) 或 Claude (https://claude.ai)
2. 点击浏览器工具栏的插件图标
3. 在弹出的窗口中：
   - 浏览 Prompt 列表
   - 使用搜索框快速查找
   - 点击任意 Prompt 卡片
4. Prompt 内容会自动填充到输入框
5. 根据需要修改后发送

## 常见问题

### Q: 测试连接失败怎么办？

A: 请检查：
1. App ID 和 App Secret 是否正确
2. 应用是否已开通多维表权限
3. 网络连接是否正常
4. 是否在飞书企业内（需要企业账号）

### Q: 点击 Prompt 后没有反应？

A: 请确认：
1. 当前页面是 ChatGPT 或 Claude
2. 页面已完全加载
3. 输入框可见且可编辑
4. 刷新页面后重试

### Q: Prompt 列表为空？

A: 请检查：
1. 多维表中是否有数据
2. Status 字段是否设置为"启用"
3. App Token 和 Table ID 是否正确
4. 点击刷新按钮重新加载

### Q: 如何更新 Prompt？

A: 直接在飞书多维表中修改即可，插件会自动同步（最多 5 分钟延迟）。也可以点击刷新按钮立即更新。
