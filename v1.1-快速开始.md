# v1.1 参数化 Prompt 快速开始

## 🚀 5 分钟上手参数化 Prompt

### 步骤 1：更新插件

1. 确保你的插件已更新到 v1.1.0
2. 在 Chrome 扩展管理页面重新加载插件

### 步骤 2：在飞书多维表中添加参数化 Prompt

打开你的飞书多维表，添加一条新记录：

| 字段 | 内容 |
|------|------|
| **Title** | 快速翻译 |
| **Content** | 请将以下内容从 [源语言] 翻译成 [目标语言]：<br><br>[待翻译内容] |
| **Category** | 工具 |
| **Status** | 启用 |

### 步骤 3：使用参数化 Prompt

1. 打开 ChatGPT 或 Claude
2. 点击浏览器右上角的插件图标
3. 点击"快速翻译"
4. 在弹出的表单中填写：
   - ⭐ 源语言：`英文`
   - ⭐ 目标语言：`中文`
   - ⭐ 待翻译内容：`Hello, how are you?`
5. 点击"确认生成并填入"
6. 完整的 Prompt 自动填入 AI 对话框！

### 步骤 4：尝试选填变量

添加另一条记录：

| 字段 | 内容 |
|------|------|
| **Title** | 邮件撰写助手 |
| **Content** | 请帮我撰写一封邮件：<br><br>收件人：[收件人]<br>主题：[邮件主题]<br>语气：[?语气风格]<br>字数：[?字数要求]<br><br>邮件内容要点：<br>[内容要点] |
| **Category** | 办公 |
| **Status** | 启用 |

使用时：
- 必填项：收件人、邮件主题、内容要点
- 选填项：语气风格、字数要求（可以留空）

## 📝 变量语法速查

### 必填变量
```
[变量名]
```
- 用户必须填写
- 表单中显示红色星号 ⭐

### 选填变量
```
[?变量名]
```
- 用户可以留空
- 留空时自动替换为"无"
- 表单中显示灰色"(选填)"

## ⚠️ 注意事项

1. **变量名要清晰**
   - ✅ `[编程语言]`
   - ❌ `[lang]`（不够清晰）

2. **合理使用必填/选填**
   - 核心信息 → 必填
   - 补充信息 → 选填

3. **控制变量数量**
   - 建议不超过 5-6 个
   - 太多会影响使用体验

4. **测试你的模板**
   - 添加后先自己测试
   - 确保各种情况都能正常工作

## 🔧 故障排除

### 问题：点击后没有弹出表单
- 检查 Prompt 中是否包含 `[变量名]` 格式
- 确保方括号是英文字符

### 问题：提示"请填写所有必填项"
- 检查是否有带红色星号的字段未填写
- 必填项不能留空

### 问题：选填项留空后显示"无"
- 这是正常行为
- 如需修改，可以调整 `popup.js` 中的替换逻辑
