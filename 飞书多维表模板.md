# 飞书多维表字段配置模板

## 字段设置

### 1. Title（单行文本）
- **字段名称**: Title
- **字段类型**: 单行文本
- **是否必填**: 是
- **说明**: Prompt 的标题或简短描述

### 2. Content（多行文本）
- **字段名称**: Content
- **字段类型**: 多行文本
- **是否必填**: 是
- **说明**: 完整的 Prompt 内容，将被填充到 AI 输入框

### 3. Category（单选）
- **字段名称**: Category
- **字段类型**: 单选
- **是否必填**: 否
- **选项配置**:
  - 研发
  - 产品
  - 设计
  - 运营
  - 客服
  - 其他

### 4. Status（单选）
- **字段名称**: Status
- **字段类型**: 单选
- **是否必填**: 否
- **默认值**: 启用
- **选项配置**:
  - 启用
  - 停用

## 使用建议

1. **标题命名**：使用简短、描述性的标题，便于快速识别
2. **内容编写**：
   - 使用清晰的指令语言
   - 可以包含占位符（如 [代码]、[需求]）
   - 分点列出要求，提高可读性
3. **分类管理**：根据团队实际情况调整分类
4. **状态控制**：使用"停用"隐藏过时的 Prompt，而不是删除
5. **持续优化**：根据使用反馈不断改进 Prompt 质量
