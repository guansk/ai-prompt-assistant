# 隐私政策

最后更新日期：2024年3月5日

## 概述

Team Prompt Manager（以下简称"本插件"）尊重并保护用户隐私。本隐私政策说明了本插件如何收集、使用和保护您的信息。

## 信息收集

### 我们不收集的信息

本插件**不收集**以下任何信息：
- ❌ 个人身份信息
- ❌ 浏览历史
- ❌ 使用统计数据
- ❌ 分析数据
- ❌ Cookie 或追踪信息
- ❌ 您在 AI 平台上的对话内容

### 本地存储的信息

本插件仅在您的本地浏览器中存储以下配置信息：
- ✅ 飞书应用凭证（App ID, App Secret）
- ✅ 飞书多维表链接（App Token, Table ID）
- ✅ Prompt 数据缓存（从飞书多维表读取）
- ✅ API 域名偏好设置

**重要说明**：
- 所有数据仅存储在您的本地浏览器中（`chrome.storage.local`）
- 不会上传到任何服务器
- 不会与第三方分享
- 您可以随时清除这些数据

## 数据使用

### 飞书 API 访问

本插件会访问飞书开放平台 API，用于：
- 获取访问令牌（tenant_access_token）
- 读取您配置的多维表数据

**数据流向**：
```
您的浏览器 → 飞书 API → 您的浏览器
```

**说明**：
- 仅在您主动操作时访问 API
- 仅读取您明确配置的多维表
- 不会修改或删除您的数据
- 不会访问其他飞书数据

### AI 平台交互

本插件会在以下 AI 平台页面注入内容脚本：
- ChatGPT (chatgpt.com, chat.openai.com)
- Claude (claude.ai)
- Gemini (gemini.google.com)
- Gemini AI Studio (aistudio.google.com)

**功能**：
- 仅用于将 Prompt 文本填充到输入框
- 不读取或记录您的对话内容
- 不修改页面的其他内容
- 不向任何服务器发送数据

## 权限说明

本插件申请以下权限：

### storage
**用途**：在本地浏览器存储配置信息
**范围**：仅限本地存储，不涉及云端

### activeTab
**用途**：获取当前标签页信息
**范围**：仅用于判断是否支持自动填充功能

### scripting
**用途**：注入内容脚本
**范围**：仅在支持的 AI 平台页面注入

### host_permissions
**用途**：访问特定网站
**范围**：
- AI 平台：用于自动填充功能
- 飞书 API：用于读取多维表数据

## 第三方服务

### 飞书开放平台

本插件使用飞书开放平台 API 读取多维表数据。

**数据处理**：
- 您的飞书凭证仅存储在本地
- API 请求直接从您的浏览器发送到飞书服务器
- 不经过任何中间服务器
- 遵循飞书的隐私政策和服务条款

**相关链接**：
- [飞书隐私政策](https://www.feishu.cn/privacy)
- [飞书开放平台文档](https://open.feishu.cn/document/)

### AI 平台

本插件在 AI 平台页面提供自动填充功能，但：
- 不收集您的对话内容
- 不向 AI 平台发送额外数据
- 不修改 AI 平台的隐私设置

## 数据安全

我们采取以下措施保护您的数据：

1. **本地存储**：所有敏感信息仅存储在本地
2. **HTTPS**：所有 API 请求使用 HTTPS 加密
3. **最小权限**：仅申请必要的权限
4. **开源代码**：代码公开透明，可审计
5. **无追踪**：不使用任何分析或追踪工具

## 数据删除

您可以随时删除本插件存储的所有数据：

### 方法一：卸载插件
1. 访问 `chrome://extensions/`
2. 找到 Team Prompt Manager
3. 点击"移除"

### 方法二：清除浏览器数据
1. 按 `Ctrl + Shift + Delete`
2. 选择"扩展程序的 Cookie 和其他网站数据"
3. 点击"清除数据"

### 方法三：手动清除
在插件的开发者工具中执行：
```javascript
chrome.storage.local.clear();
```

## 儿童隐私
