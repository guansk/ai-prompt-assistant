# Team Prompt Manager - 安装与使用指南

## 📦 安装步骤

### 1. 准备图标文件
在 `icons/` 目录下放置三个尺寸的图标：
- icon16.png (16x16)
- icon48.png (48x48)
- icon128.png (128x128)

### 2. 加载插件到 Chrome

1. 打开 Chrome 浏览器
2. 访问 `chrome://extensions/`
3. 开启右上角的"开发者模式"
4. 点击"加载已解压的扩展程序"
5. 选择本项目的根目录

## ⚙️ 配置飞书

### 1. 创建飞书自建应用

1. 访问 [飞书开放平台](https://open.feishu.cn/app)
2. 点击"创建企业自建应用"
3. 填写应用名称和描述
4. 创建完成后，记录 **App ID** 和 **App Secret**

### 2. 配置应用权限

在应用管理页面：
1. 进入"权限管理"
2. 搜索"多维表"
3. 开通以下权限：
   - 查看、评论、编辑和管理多维表（`bitable:app`）
   - 或仅开通"查看多维表"（最小权限原则）

### 3. 创建多维表

1. 在飞书中创建一个新的多维表
2. 添加以下字段：

| 字段名 | 字段类型 | 说明 |
|--------|----------|------|
| Title | 单行文本 | Prompt 标题（必填） |
| Content | 多行文本 | Prompt 内容（必填） |
| Category | 单选 | 分类标签（可选） |
| Status | 单选 | 状态：启用/停用（可选） |

3. 添加一些示例数据：

**示例 1:**
- Title: Python 代码审查
- Content: 请审查以下 Python 代码，关注代码质量、性能优化和最佳实践，并提供改进建议。
- Category: 研发
- Status: 启用

**示例 2:**
- Title: 产品需求分析
- Content: 请帮我分析以下产品需求，包括用户痛点、核心价值、技术可行性和潜在风险。
- Category: 产品
- Status: 启用

### 4. 获取多维表标识

从多维表的 URL 中提取信息：
```
https://example.feishu.cn/base/[App Token]?table=[Table ID]&view=xxx
```

- **App Token**: `bascn` 开头的字符串
- **Table ID**: `tbl` 开头的字符串

## 🎯 使用插件

### 1. 配置插件

1. 点击浏览器右上角的插件图标
2. 点击"设置"按钮（齿轮图标）
3. 填写飞书配置信息：
   - App ID
   - App Secret
   - App Token
   - Table ID
4. 点击"测试连接"验证配置
5. 点击"保存配置"

### 2. 使用 Prompt

1. 打开 ChatGPT (chatgpt.com) 或 Claude (claude.ai)
2. 点击浏览器右上角的插件图标
3. 在弹出窗口中：
   - 浏览 Prompt 列表
   - 使用搜索框快速查找
   - 点击任意 Prompt 自动填充到输入框

### 3. 管理 Prompt

直接在飞书多维表中：
- 添加新的 Prompt
- 修改现有内容
- 设置分类标签
- 停用不需要的 Prompt

插件会自动同步最新数据（5分钟缓存）。

## 🔧 故障排查

### 问题：无法连接飞书 API

**解决方案：**
1. 检查 App ID 和 App Secret 是否正确
2. 确认应用已开通多维表权限
3. 检查网络连接

### 问题：找不到输入框

**解决方案：**
1. 确认当前页面是 ChatGPT 或 Claude
2. 刷新页面后重试
3. 如果页面改版，可能需要更新插件

### 问题：Prompt 列表为空

**解决方案：**
1. 检查多维表中是否有数据
2. 确认 Status 字段设置为"启用"
3. 验证 App Token 和 Table ID 是否正确

## 🎨 UI 特点

- **简洁高级**：采用现代化设计语言，清爽的配色方案
- **流畅交互**：平滑的动画过渡，即时的视觉反馈
- **响应式布局**：适配不同屏幕尺寸
- **状态管理**：清晰的加载、错误、空状态提示

## 📝 注意事项

1. **安全性**：App Secret 仅存储在本地浏览器，不会上传到任何服务器
2. **权限**：建议仅授予"读取"权限，防止数据被意外修改
3. **缓存**：数据缓存 5 分钟，可手动点击刷新按钮更新
4. **兼容性**：目前支持 ChatGPT 和 Claude，其他网站可能不兼容
