## 参数化 Prompt (v1.1)

### 什么是参数化 Prompt？

参数化 Prompt 允许你在模板中定义变量，使用时填写变量值，让一个模板适配多种场景。

### 如何识别参数化 Prompt？

在 Prompt 列表中，参数化的 Prompt 会显示蓝色徽章，如：

```
┌─────────────────────────────┐
│ 快速翻译  [3 个参数]  工具  │
│ 请将以下内容从...           │
└─────────────────────────────┘
```

### 使用步骤

#### 步骤 1：点击参数化 Prompt

点击带有参数徽章的 Prompt。

#### 步骤 2：填写表单

弹出的表单会显示所有需要填写的变量：

```
┌─────────────────────────────┐
│ 填写参数                     │
│ ℹ️ 带 * 为必填项，其他为选填  │
├─────────────────────────────┤
│ * 源语言                     │
│ ┌─────────────────────────┐ │
│ │ 请输入...               │ │
│ └─────────────────────────┘ │
│                             │
│ * 目标语言                   │
│ ┌─────────────────────────┐ │
│ │ 请输入...               │ │
│ └─────────────────────────┘ │
│                             │
│ 专业领域 (选填)              │
│ ┌─────────────────────────┐ │
│ │ 选填，可留空            │ │
│ └─────────────────────────┘ │
├─────────────────────────────┤
│ [返回列表] [确认生成并填入]  │
└─────────────────────────────┘
```

#### 步骤 3：提交表单

- **必填项**：带红色星号 ⭐，必须填写
- **选填项**：灰色"(选填)"提示，可以留空
- 点击"确认生成并填入"

#### 步骤 4：自动填入

完整的 Prompt 会自动填入 AI 对话框，可以直接发送。

### 变量语法说明

如果你需要在飞书多维表中创建参数化 Prompt：

| 语法 | 类型 | 说明 | 示例 |
|------|------|------|------|
| `[变量名]` | 必填 | 用户必须填写 | `[编程语言]` |
| `[?变量名]` | 选填 | 用户可以留空 | `[?代码功能]` |

### 实际示例

#### 示例 1：翻译助手

**配置：**
```
请将以下内容从 [源语言] 翻译成 [目标语言]：

专业领域：[?专业领域]

[待翻译内容]
```

**使用：**
1. 点击"翻译助手"
2. 填写：
   - 源语言：`英文`
   - 目标语言：`中文`
   - 专业领域：留空或填写 `医学`
   - 待翻译内容：粘贴要翻译的文本
3. 点击确认

**结果：**
```
请将以下内容从 英文 翻译成 中文：

专业领域：医学

[你粘贴的文本]
```

#### 示例 2：代码审查

**配置：**
```
请审查以下代码：

编程语言：[编程语言]
代码功能：[?功能描述]

[代码内容]

请提供改进建议。
```

**使用：**
1. 点击"代码审查"
2. 填写：
   - 编程语言：`Python`
   - 功能描述：`用户登录验证`（选填）
   - 代码内容：粘贴代码
3. 点击确认

## 常见问题

### Q1: 如何知道哪些 Prompt 是参数化的？

A: 参数化的 Prompt 会在标题旁显示蓝色徽章，如"3 个参数"。

### Q2: 必填项没填会怎样？

A: 系统会阻止提交，并在未填写的输入框显示红色边框提示。

### Q3: 选填项留空会怎样？

A: 选填项留空时，会自动替换为"无"，保证 Prompt 语句通顺。

### Q4: 可以返回修改吗？

A: 可以。点击"返回列表"按钮即可返回，不会提交。

### Q5: 无变量的 Prompt 还能用吗？

A: 完全可以！无变量的 Prompt 保持原有的一键填充行为。

### Q6: 如何在飞书多维表中创建参数化 Prompt？

A: 在 Content 字段中使用 `[变量名]` 或 `[?变量名]` 语法即可。详见设置页面的使用说明。

## 使用技巧

### 技巧 1：善用搜索

在搜索框输入关键词，快速找到需要的 Prompt。搜索支持标题和内容的模糊匹配。

### 技巧 2：查看参数数量

点击前先看参数徽章，了解需要填写多少个变量，做好准备。

### 技巧 3：选填项灵活使用

选填项可以根据实际需求填写或留空，不必每次都填。

### 技巧 4：复制粘贴长文本

对于长文本变量（如代码、文章），建议先在编辑器中准备好，再复制粘贴到表单。

### 技巧 5：查看设置页面

点击插件右上角的齿轮图标，可以查看详细的参数化 Prompt 使用说明和示例。

### 技巧 6：刷新数据

如果飞书多维表中的数据更新了，点击刷新按钮获取最新数据。
